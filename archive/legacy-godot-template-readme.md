# Legacy Godot Template README

This repository originally contained a Godot tactical RPG template. The current CCZ engine direction is Android/Kotlin-first, but the Godot material remains as reference/prototype content.

Original upstream-style summary:

- Godot Engine 4 tactical RPG template.
- Turn-based grid movement.
- Pawn movement and attack.
- Basic enemy AI.
- Camera panning, free look, zoom and rotations.
- Blender map recognition tutorial under `docs/tutorials/how-to-create-maps/`.
- Controller support.
- Godot editor documentation and debugger support.

Relevant current files:

```text
project.godot
assets/
data/
docs/tutorials/
docs/img/
```
